# Docker Compose 사용하기 

## 실행하기

```
docker-compose up -d
```

## 로그 보기

```
docker-compose logs -f
```

## 종료하기

```
docker-compose down
```

볼륨도 함께 삭제하고 싶다면

```
docker-compose down --volume
```
